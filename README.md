# Zambezi Strategy Group

Pan-African luxury brand strategy consultancy website.

## Setup

```bash
npm install
npm start
```

## Deploy to Netlify

```bash
npm run build
```

Then drag the `build` folder into Netlify, or connect your GitHub repo directly.
